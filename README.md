# Git Workflow

1. Create new branch from "master"
    git checkout -b name-of-issue
2. Push your changes to the branch
3. Checkout to branch "dev"
4. Merge branch 
    (git merge name-of-issue)
5. Push to branch
    (git push origin dev)
7. Assign issue as ready for testing
